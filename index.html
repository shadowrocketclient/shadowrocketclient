Frequently Asked Questions (FAQ)

How to import server configurations?
Tap the "+" button on the top right of the home page, select "Scan QR Code" or "Import from Clipboard" to quickly add server configurations. We support various common subscription formats.
What if the connection fails?
First, please check if your network connection is normal. Second, tap "Connectivity Test" to check the node status. If it shows all red, please check the configuration information (port, password, encryption method) or contact your service provider.
How to customize routing rules?
Go to the "Config" page, tap on the current configuration file to edit it, or import a remote configuration file. We support adding custom rules such as domains, IP ranges, and more.
